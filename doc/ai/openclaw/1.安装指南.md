## 注册 MiniMax

1. **不要开科学上网**，-> **浏览器访问[minimaxi](https://www.minimaxi.com/)**
2. **开放平台** → **平台登陆** → 使用手机号登陆
3. **账户管理** → **创建新的 api key** → 将密钥粘贴到临时位置，后续会用到
4. **财务管理** → **余额** 中可查看赠送的 15 元代金券（暂时够用，后续如需可在此充值）
5. **套餐管理** → **coding plan** 中可看到已自动激活试用版，注意可用额度和截止日期

## 安装 OpenClaw

1. **科学上网**后访问：[openclaw官方指引文档](https://docs.openclaw.ai/start/getting-started)（以下步骤参考该指引）
2. **mac**：在终端 zsh 中执行  
   `curl -fsSL https://openclaw.ai/install.sh | bash`  
   **Windows**：以管理员身份打开 PowerShell，执行  
   `iwr -useb https://openclaw.ai/install.ps1 | iex`
3. 执行 `openclaw onboard --install-daemon`  
   - 仅选择模型为 **minimax**，其余先跳过，后续再配置  
   - **不要选飞书**，飞书会在后面单独配置

### 2. 安装飞书插件

**Linux / macOS：**

```bash
npm config set registry https://registry.npmjs.org
curl -o /tmp/feishu-openclaw-plugin-onboard-cli.tgz https://sf3-cn.feishucdn.com/obj/open-platform-opendoc/4d184b1ba733bae2423a89e196a2ef8f_QATOjKH1WN.tgz
npm install /tmp/feishu-openclaw-plugin-onboard-cli.tgz -g
rm /tmp/feishu-openclaw-plugin-onboard-cli.tgz
```

**Windows（CMD）：**

```bash
npm config set registry https://registry.npmjs.org
curl -o "%TEMP%\feishu.tgz" https://sf3-cn.feishucdn.com/obj/open-platform-opendoc/c53145d7b9eb0e29f4e07bf051231230_XjCy46mAFI.tgz
npm install -g "%TEMP%\feishu.tgz"
del "%TEMP%\feishu.tgz"
```

安装完成后执行：

```bash
feishu-plugin-onboard install
```

若之前已关联过飞书应用，可选择沿用或创建/关联新应用；若创建或关联新应用，需填入上一步的 **App ID** 和 **App Secret**。

## 运行 OpenClaw

1. 启动网关：

   ```bash
   openclaw gateway run
   ```

   出现监听飞书事件的提示即表示启动成功。

2. 检查插件状态：

   ```bash
   openclaw plugins list
   ```

   - `feishu-openclaw-plugin` 的 Status 为 **loaded**  
   - `feishu` 的 Status 为 **disabled**  
   即表示飞书官方插件已正确启用。

3. **绑定机器人**  
   - 在飞书中给机器人发任意消息，系统会生成一个配对码（字母+数字）。  
   - 若之前安装过飞书插件，可能不会再次出现配对流程。

4. 在终端执行绑定（将 `<配对码>` 替换为实际配对码）：

   ```bash
   openclaw pairing approve feishu <配对码> --notify
   ```

## 开启openclaw工具
1. 访问[openclaw工具指南](https://docs.openclaw.ai/zh-CN/tools)
2. 找到希望开启的工具的名称
3. 打开[可视化配置页面](http://127.0.0.1/config)
4. 点击Tools -> 找到 Tool Allowlist -> 填入你希望开启的工具
